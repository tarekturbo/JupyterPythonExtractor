# webapp to Extract python code form Jupyter file
## Installing
Please install the requirements:
```
# Linux/macOS
python3 -m pip install -r requirements.txt
# Windows
py -3 -m pip install -r requirements.txt
```
## Runing 
```
python app.py
```
